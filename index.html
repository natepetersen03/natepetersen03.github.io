Projects

MacroMeals

MacroMeals is a full-stack web application containing a unique dataset of over 27,000 recipes and a comprehensive React based UI that allows users to search for recipes that fit nutrient specifications. To create the dataset, I merged a recipe dataset, RecipeNLG, with a nutritional dataset using regular expression parsing and string matching techniques to provide users with recipe instructions, ingredients, and their associated nutrient profiles. After the dataset was finished, I used Python to implement the backend, leveraging SQLAlchemy to query a MySQL database containing recipe data and Flask to develop API endpoints.

Government Spending Forecaster

Created a predictive model for stock price forecasting based on data from presidential election cycles through working with team members. In doing this, I used natural language processing and sentiment analysis techniques to analyze 8000+ lines of presidential debate transcripts. After the dataset was annotated, I effectively implemented machine learning models, specifically utilizing BERT and PyTorch in coordination with the annotated dataset to produce comprehensive stock price forecasts.

FroggerDroid

FroggerDroid is a unique version of Frogger featuring Minecraft skins and entities. The game features a home screen where users get to choose their characters and game difficulty, a main screen where users get to play the game, and an end screen for when the game is finished. To build the Android app, I used Gradle and coded in Java within Android Studio, integrating the LibGDX library.
